# LOB SDK

**LOB SDK** is a TypeScript library that provides core game logic, data structures, and utilities for building applications that interact with [Lines of Battle](https://linesofbattle.net). It includes game data management, AI bot implementations, pathfinding algorithms, and type definitions.

## Install

Add the SDK directly from GitHub in your `package.json`:

```json
"dependencies": {
  "lob-sdk": "git+https://github.com/sophie-games/lob-sdk.git"
}
```

To pin to a specific commit SHA:

```json
"dependencies": {
  "lob-sdk": "git+https://github.com/sophie-games/lob-sdk.git#<commit-sha>"
}
```

Or install via CLI:

```bash
npm install git+https://github.com/sophie-games/lob-sdk.git
# or pinned
npm install git+https://github.com/sophie-games/lob-sdk.git#<commit-sha>
```
